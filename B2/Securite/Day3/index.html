Vocabulaire

Vulnérabilité

Un bug dans un logiciel

Payload

La charge utile qui va provoquer/exploiter ce bug

Shellcode

Chaine de caractères représentant un code executable

C'est l'action qui va etre executé quand on exploite la vuln

Dans un Payload on peux ouvrir une connexion vers l'attaquant

2 types de connexion :

Bind : ouvre un port sur la victime

Connect Back : se connecte à l'attaquant

On peux aussi faire tout un tas de trucs

Créer un compte, lire un fichier, upload/download
