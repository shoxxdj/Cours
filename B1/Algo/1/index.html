Les boucles

Pourquoi répéter bêtement 100x la meme chose

Chaque boucle utilise un compteur

Ce compteur est en réalité une variable

Tant que

Cette boucle s'execute tant que la condition n'est pas remplie

ENTIER i = 0;
TANT QUE(i<10){
	Afficher(i);
	i+=1;
}

Le modulo

Le modulo

"Division reste" -> division Euclidienne

Le résultat de la fonction modulo correspond au reste de la division euclidienne

Exemple :

Modulo(5,2)

5/2 =2+1

Modulo(5,2)=1

Modulo(6,3)=0
