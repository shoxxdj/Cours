La Table ASCII

Votre ordinateur ne connait pas les caractères alphabetiques

Par contre il connait les nombres

La table ascii permet d'assigner une valeur à un caractère

Il existe des fonctions pour obtenir la position d'un caractère dans la table ascii

Sur linux -> man ascii

Les boucles

Tant Que

L'action s'effectue tant que la condition est respectée

L'action ne s'effectue pas forcement

Faire Tant Que

L'action s'effectue au moins une fois

La condition est vérifiée à la fin du premier tour de boucle

Entier i=0
Faire{
	i+=1
} TantQue(i<10)

Pour

L'action s'effectue un nombre déterminée de fois

Pour(i;i<10;i++){
	Afficher(i)
}
		    		
Pour(ENIER a=20;a<10;a--){
	Afficher(a)
}

Les Tableaux

2 dimensions

Lignes / Colones

== aux tableaux a plusieures dimensions

Il faut cependant préciser le nombres de lignes

Char MonTableau[3][3]=[['a','b','c'],['d','e','f'],['g','h','i']]
Afficher(MonTableau[0][0]) #Affiche 'a'
Afficher(MonTableau[2][1]) #Affiche 'h'
MonTableau[1][2]='z'

Exo ?

Créer un tableau de 9 lignes / colones

Pour chaque case l'utilisateur entre un nombre

Une fois l'initialisation terminée, pour chaque ligne l'utilisateur peux:

  • Additionner tout le contenu de la ligne et avoir le résulat
  • Multiplier tout le contenu de la ligne et avoir le résultat
  • Ne rien faire et passer à la ligne suivante
Pensez aux fonctions :)
