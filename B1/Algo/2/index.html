Exo 10

Nombres premiers

Lire un nombre donné par l'utilisateur, déterminer si il est premier ou non

Un nombre premier n'est divisible que par 1 et lui meme

Entier Nombre=Lire()
Entier Diviseur
Boolean EstPremier

Diviseur = Nombre-1
EstPremier = Vrai
Tantque(EstPremier && Diviseur>=2){
	Si(MOD(Nombre,Diviseur)==0){
		EstPremier=Faux
	}
	Diviseur--
}

Si(EstPremier){
	Afficher("Nombre Premier")
}
Sinon{
	Afficher("Non Premier")
}
