Fil Rouge

Le but de ce TP est de vous faire pratiquer le scanning. Le TP n'est pas très long, mais prennez bien le temps de comprendre ce que vous faites.

  • Téléchargez et mettez en place la machine virtuelle Metasploitable
  • Vérifiez que la machine est démarée et accessible dans votre réseau interne
  • A l'aide de nmap
    • Trouvez les ports ouverts sur la machine
    • Trouvez l'OS et la version de l'os installé sur la machine
  • Installez Nessus ( Privilégiez la version Linux )
    • Rendez vous sur le site de la boite propriétaire de Nessus
    • Téléchargez la version Home (n'hesitez pas à utiliser des mails anonymes)
    • A vous de trouver comment installer nessus :) Lien Magique
  • Utilisez Nessus
    • Scannez la machine Metasploitable
    • Que ce passe t il pendant le scan ?
    • Quels sont les résultats ?
    • Pour chaque résultat Critique, High, et Medium : Qualifiez le résultat, tentez de le vérifier ( si il s'agit d'un problème de version essayez de retrouver cette version, vous pouvez acceder à la machine)
    • Pour les résultats contenant des CVE retrouvez la CVE concerné. Expliquez ce qu'est une CVE et son utilitée
  • Il existe d'autres scanners que Nessus, trouvez en un, téléchargez le et utilisez le sur la machine. Comparez les résultats. Pourquoi les résultats sont ils différents ?
