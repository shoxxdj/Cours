Fil Rouge

Le but de ce TP est de vous faire pratiquer le footprinting. Vous allez donc devoir trouver des informations sur divers supports. Pour chaque question, vous répondrez en expliquant ce que vous avez fait et prendrez si nécéssaire des screenshots.

Vous pouvez le faire à plusieurs mais un rendu est attendu par étudiant.

Evidemment vous n'avez pas toutes les réponses, et ne savez pas comment marchent les outils, pour plus d'informations voyez ici

  1. Trouvez l'ip du domaine ynov.com
  2. Sur un linux, en CLI ( Console ) trouvez les informations suivantes :
    1. IP du site hackdumb.com
    2. Adresse MX du site hackdumb.com
  3. Affichez le Whois de Hackdumb.com.
    1. Quand le nom de domaine a-t-il été créé ?
    2. Quand a-t-il été mis à jour pour la denière fois ?
  4. Retrouvez la première page publiée sur Hackdumb.com Pensez aux réseaux sociaux
  5. Depuis votre navigateur affichez les headers de ynov.com. Que constatez vous ?
  6. Depuis telnet affichez les entetes de ynov.com. Que constatez vous ?
  7. A l'aide de netcraft.com trouvez les urls associées au domaine ynov.com. La liste est-elle complète, trouvez un moyen de compléter cette liste.
  8. Trouvez les définitions des mots suivants :
    1. Eavesdropping
    2. Shoulder Surfing
    3. Dumpster Diving
  9. A l'aide de google Maps, trouvez les serveurs d'ingésup Toulouse.
  10. A l'aide de google trouvez :
    1. Des documents confidentiels
    2. Des FTP ouverts
    3. Des fichiers excels
  11. Grâce à httrack, copiez le site sso.ynov.com. Hébergez le sur un serveur local. Quel est l'utilitée de cette manipulation ?
  12. Rendez vous sur les réseaux sociaux, essayez d'en apprendre le plus possible sur l'organigrame d'ynov.
