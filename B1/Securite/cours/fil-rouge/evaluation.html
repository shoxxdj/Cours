Fil Rouge
Evaluation du cours / TP fil rouge

Voici venu l'heure des notes ! afin d'etre le plus juste possible les notes seront établies comme suit :

    Cours :
  • Une première note sur l'ensemble des TP que vous avez rendu
    • TP Fil Rouge :
  • /10 Une note sur l'avancée dans le projet. Pour info les critères étaient dispo sur la page de présentation du projet :)
  • /5 Une note sur la partie défense, les différents mécanismes mis en place
  • /5 Une note sur la partie attaque, vos réalisations ..
  • /20 Une note concernant votre présentation
      Points à aborder dans la présentation:
    • Rappel du projet
    • Votre organisation ( qui a fait quoi )
    • Ce que vous avez produit ( presentation du blog )
    • Les attaques que vous avez réalisé
    • Ce que vous avez subit et comment vous auriez pu l'empecher
  • /20 Une note sur le partiel :
      Organisation du partiel :
    • Vous aurez une séries d'épreuves traitant des différents sujets abordés en cours
    • Chaque épreuve vous rapportera des points.
    • Vous devez chacun écrire un writeup* par épreuve. ( pensez à prendre des screenshots )
    • Le writeup doit etre lisible par des non initiés
    • Vous avez 4h pour faire ce partiel
    • *writeup -> Document explicant ce que vous avez trouvé et comment vous etes arrivé au flag.
